# Master Memory

Project Brain's compressed project intelligence. This is the first file read by AI agents.

## Project Summary

Project Brain is a persistent intelligence layer for AI-powered software development. It builds and maintains a compressed understanding of the codebase in the `brain/` directory to serve as long-term memory for AI agents.

## Core Value

Continuously build, maintain, and evolve a compressed, high-fidelity project understanding to reduce token waste by 70-95% and ensure architectural consistency.

## Architecture Summary

```
project/
├── brain/          # Memory layer
├── skills/         # Executable agent skills
├── workflows/      # Agent workflow specifications
└── source-code/    # Core application files
```

- **Runtime**: Google Antigravity / Gemini CLI
- **Agents**: Direct Antigravity local skills (Python scripts)
- **State tracking**: Automated task output parsing and documentation updates
