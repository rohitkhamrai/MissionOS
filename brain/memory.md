# Project Memory

Central project summary and status board.

## What the Project Does

Project Brain provides a persistent intelligence layer for AI-powered software development. It maintains a curated knowledge system that summarizes the project, eliminating the need for AI agents to repeatedly scan the entire codebase.

## Key Features

- **Compressed Memory**: High-fidelity summaries in `master-memory.md` under 5,000 words.
- **Dynamic Updates**: Automatically updates after successful tasks via a dedicated Memory Agent.
- **Structured Knowledge**: Categorized files tracking architecture, patterns, decisions, mistakes, and roadmap.
