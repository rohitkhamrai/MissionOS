# Glossary

Project terminology and definitions.

## Definitions

- **Project Brain**: The persistent intelligence layer maintaining codebase state in the `brain/` directory.
- **Memory Agent**: The automation agent responsible for parsing git commits, walkthroughs, and logs to update the project memory.
- **Master Memory**: A highly compressed version of all memory files kept under 5,000 words to minimize token waste.
- **Local Skill**: Custom agent automation packaged in `.agents/skills` for execution by Google Antigravity / Gemini.
